NAME
====

**echo** - print arguments to stdout

SYNOPSIS
========

| **echo** [-n] [_ARGUMENTS_]

DESCRIPTION
===========

Prints the arguments given via stdin to stdout.

FLAGS
=====

**-n**  Omit trailing newlines.
