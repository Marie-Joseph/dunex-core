NAME
====

**mkfifo** - create a new named pipe (fifo).

SYNOPSIS
========

| **mkfifo** [**-m**|**\--mode** _mode_] _PATH_

DESCRIPTION
===========

Creates a new named pipe (fifo).

OPTIONS
=======

**-m**, **\--mode** *mode*  Specify the mode of newly created pipe (default is 0666 & (~umask)).
