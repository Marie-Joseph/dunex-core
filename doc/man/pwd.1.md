NAME
====

**pwd** - print the present working directory

SYNOPSIS
========

| **pwd** [_FLAGS_]

DESCRIPTION
===========

Prints the present working directory.

FLAGS
=====

**-L**  Print the logical present working directory (default).

**-P**  Print the physical working directory, with symlinks resolved.
